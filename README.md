# 厚墨App书源整理

**提供最好用的版本**：
1. 最初的神版，也是最后的神版。
2. 没有2.0的书籍限制，有书源就可以搜索到。

**下载地址**：
[厚墨_1.3.0.apk](https://tool.double71.com/houmo/厚墨_1.3.0.apk)

**说明书：** 首先复制链接：
[书源地址](https://cdn.jsdelivr.net/gh/double71/BookSourceRepository@master/repository.json)

**打开app > 头像 > 书源 > 右上角的 + 号> 网址导入 > 输入链接 > 验证 > 完成**
